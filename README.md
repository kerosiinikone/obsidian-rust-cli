# obsidian-rust-cli (in prog.)

Create a new idea note / append to notes from a "fleeting thought". Open, create (daily) notes. Vault statistics. _Multiple interfaces._

### Commands (plain CLI)

Check out `Makefile`.

### Commands (with TUI)
